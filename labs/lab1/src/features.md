# features.md

## Описание признаков (начальный этап)

| Поле            | Описание                                                                 | Исходный тип данных | Польза для модели | Предобработка | Выходной тип данных |
|-----------------|---------------------------------------------------------------------------|----------------------|-------------------|----------------|----------------------|
| `id`            | Уникальный идентификатор продажи                                         | int                  |                   |                |                      |
| `date`          | Дата продажи                                                             | object (string)      |                   |                |                      |
| `price`         | Цена продажи (целевой признак)                                           | float                |                   |                |                      |
| `bedrooms`      | Количество спален                                                        | int                  |                   |                |                      |
| `bathrooms`     | Количество ванных комнат                                                 | float                |                   |                |                      |
| `sqft_living`   | Жилая площадь                                                             | int                  |                   |                |                      |
| `sqft_lot`      | Площадь участка                                                           | int                  |                   |                |                      |
| `floors`        | Количество этажей                                                        | float                |                   |                |                      |
| `waterfront`    | Признак выхода к воде (0/1)                                              | int                  |                   |                |                      |
| `view`          | Оценка вида (0–4)                                                        | int                  |                   |                |                      |
| `condition`     | Оценка состояния (1–5)                                                   | int                  |                   |                |                      |
| `sqft_above`    | Площадь над уровнем земли                                                | int                  |                   |                |                      |
| `sqft_basement` | Площадь подвала                                                          | int                  |                   |                |                      |
| `yr_built`      | Год постройки                                                            | int                  |                   |                |                      |
| `yr_renovated`  | Год последнего ремонта                                                   | int                  |                   |                |                      |
